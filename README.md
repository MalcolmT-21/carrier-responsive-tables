# Responsive Status Codes Reference Tables

## Overview

This project recreates technical reference tables that were originally provided as an SVG image. The goal was to make the content easier to read and use across desktop, tablet, and mobile devices while preserving the original information.

The solution was built using semantic HTML and Bootstrap 5.

## Approach

The original source was a static SVG containing multiple technical tables. Rather than displaying the SVG directly, the content was recreated as structured HTML to improve accessibility, readability, and responsiveness.

Key decisions included:

* Converting image-based content into HTML tables and content sections.
* Using Bootstrap's grid system to support responsive layouts.
* Splitting large datasets into smaller sections to improve readability on larger screens.
* Preserving the original technical information while presenting it in a more user-friendly format.
* Organizing content into logical sections including Status Codes, Menu Navigation, and Component Tests.

## Responsive Design Considerations

### Desktop and Tablet

* Technical data is displayed in table format for easy scanning and comparison.
* Large tables are organized into multiple sections to reduce excessive horizontal width.
* Bootstrap's responsive grid system is used to improve layout and readability.

### Mobile

* Content is structured to remain readable on smaller screens.
* Tables are designed to adapt to narrower viewports without requiring users to view the original image.
* Information is organized to prioritize readability and usability on mobile devices.

## Tradeoffs

* The original SVG content was manually recreated as HTML.
* Some layout decisions were adjusted to improve readability and responsiveness across devices.
* The focus of the solution was usability and accessibility rather than creating an exact visual replica of the SVG.

## What I Would Add With More Time

* Expandable mobile sections for large data sets.
* Additional styling refinements and visual polish.
